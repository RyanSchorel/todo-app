Infinite scroller

In module 7 van het BO-programma (BO = beroeps opdracht) heb ik een project gemaakt genaamd "Infinite Scroll". Dit project maakt gebruik van verschillende technieken zoals Javascript, JSON, CSS, HTML en Python. De essentie van dit project is om een website te ontwikkelen die een eindeloze scroll aan contentberichten. Dit heb ik gemaakt als de lay out van Instagram en de stijl van Twitch. Het doel van dit project was om een nooit eindigende scrollervaring te creëren, waarbij de gebruiker continu nieuwe content kan zien zonder handmatig te hoeven vernieuwen. Dit project is gemaakt door Ryan Schorel (doormij dus), en ik ben blij met het eindresultaat. Een interessante uitdaging die me heeft geholpen mijn programmeervaardigheden verder te ontwikkelen.
